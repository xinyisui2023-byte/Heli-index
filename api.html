📊 指数数据接口

获取合力指数系列行情、成分股、历史数据等

GET /v1/indices/quotes
获取合力300、港股通新质、纳斯达克中国新质三大指数实时行情
Query Parameters
参数类型说明
codes 必填string指数代码，逗号分隔。如：HELI300,HSINEW,NASDQCN
fields 可选string返回字段，默认全部。可选：price,change,volume,turnover
// 请求示例 curl -H "Authorization: Bearer YOUR_API_KEY" \ "https://api.heli-index.com/v1/indices/quotes?codes=HELI300,HSINEW" // 响应示例 { "code": 0, "data": [ { "code": "HELI300", "name": "合力300", "price": 4821.37, "change": 52.18, "changePct": 1.09, "volume": 84230000000, "updateTime": "2026-07-06T14:30:00+08:00" } ] }
GET /v1/indices/{code}/constituents
获取指定指数的成分股列表及热力值排名
// 请求示例 curl -H "Authorization: Bearer YOUR_API_KEY" \ "https://api.heli-index.com/v1/indices/HELI300/constituents?sort=heat&limit=10"
GET /v1/indices/{code}/history
获取指数历史K线数据，支持日/周/月线
// 请求示例 curl -H "Authorization: Bearer YOUR_API_KEY" \ "https://api.heli-index.com/v1/indices/HELI300/history?period=day&from=2026-01-01&to=2026-07-06"

🔥 热力值与共识数据

获取热力值排名、共识度分析、投资者情绪等数据

GET /v1/heatmap/ranking
获取全市场热力值排名，支持按行业、市值筛选
GET /v1/consensus/{symbol}
获取指定股票的投资者共识度数据（投票分布、分歧度等）
// 请求示例 curl -H "Authorization: Bearer YOUR_API_KEY" \ "https://api.heli-index.com/v1/consensus/300750?diagnosis=2026Q3" // 响应示例 { "symbol": "300750", "name": "宁德时代", "consensusScore": 92.4, "votes": { "bullish": 782, "neutral": 156, "bearish": 47 }, "divergence": 0.12 }
GET /v1/diagnosis/reports
获取诊断季报告数据，含公司得分、投资者共识、风险提示
