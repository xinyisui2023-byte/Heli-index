🤝 什么是"共识驱动"？

合力300是全球首只将投资者共识纳入指数编制核心因子的基准指数。每季度通过诊断季活动和投资者投票，动态调整"热力值"——反映市场对新质生产力公司的真实认可度。

热力值 = 投票热度(40%) + 提问质量(30%) + 共识偏差度(30%)
热力值越高 → 成分股权重上调±2%（上限）

因子1：创新投入因子 — 权重25%

衡量企业持续创新的基础投入强度，是MSCI因子体系中"Quality"因子的中国版本。
F₁ = w₁₁×R&D/Sales + w₁₂×研发人员占比 + w₁₃×专利密度
w₁₁=0.55, w₁₂=0.25, w₁₃=0.20
门槛值：R&D/Sales≥5% 或研发人员≥15%
评分细则：R&D≥15%→10分 | 10%-15%→8分 | 5%-10%→6分
数据来源：公司年报研发费用明细 + 国知局专利数据
▶ 展开子指标
子指标1：研发费用率(55%)—研发费用/营业收入。制造业纳入资本化研发支出。
子指标2：研发人员占比(25%)—研发人员/总员工数。≥30%优秀，≥15%合格。
子指标3：专利密度(20%)—近3年发明专利数/研发投入(亿元)。反映研发产出效率。
🤝

因子2：共识度因子 — 权重20%

合力300的核心创新因子，将投资者共识量化纳入指数编制，全球首创。
F₂ = w₂₁×投票一致性 + w₂₂×分析师分歧度 + w₂₃×热力值排名
w₂₁=0.35, w₂₂=0.30, w₂₃=0.35
无门槛值（所有成分股均参与评分）
评分细则：投票一致性≥80%→10分 | 60%-80%→7分 | <60%→4分
数据来源：诊断季投票数据 + 分析师评级 + 热力值引擎
▶ 展开子指标
子指标1：投票一致性(35%)—投资者投票中，看好/中立/看空三选项的一致程度。高度一致=共识强。
子指标2：分析师分歧度(30%)—分析师评级(买入/持有/卖出)的标准差。分歧大=共识弱。
子指标3：热力值排名(35%)—合力指数平台热力值排名百分位。TOP10%=10分，TOP50%=5分。
🌱

因子3：绿色ESG因子 — 权重15%

衡量企业可持续发展能力，参照MSCI ESG评级框架，中国本土化适配。
F₃ = w₃₁×碳排放强度 + w₃₂×MSCI ESG评级 + w₃₃×绿色专利数
w₃₁=0.35, w₃₂=0.40, w₃₃=0.25
淘汰值：MSCI ESG < BBB
评分细则：ESG AAA→10分 | AA→8分 | A→6分 | BBB→4分 | 数据来源：MSCI ESG评级 + 公司ESG报告 + 碳披露数据
📊

因子4：成长质量因子 — 权重15%

衡量企业成长的真实性和可持续性，对应MSCI Growth因子。
F₄ = w₄₁×营收CAGR + w₄₂×ROE + w₄₃×毛利率趋势
w₄₁=0.4, w₄₂=0.35, w₄₃=0.25
门槛值：营收CAGR≥10%, ROE≥12%
评分细则：CAGR≥30%→10分 | 20%-30%→8分 | 10%-20%→6分
数据来源：公司年报 + Wind财务数据
💻

因子5：数字智能因子 — 权重15%

衡量企业数字化转型与AI应用水平，合力指数独创因子。
F₅ = w₅₁×数字化收入占比 + w₅₂×AI应用评分 + w₅₃×数据资产规模
w₅₁=0.45, w₅₂=0.35, w₅₃=0.20
门槛值：数字化收入占比≥10%
评分细则：数字化占比≥40%→10分 | 20%-40%→7分 | 10%-20%→4分
数据来源：公司年报 + 合力指数AI分类引擎验证
🏭

因子6：产业链安全因子 — 权重10%

衡量企业在国产替代和供应链自主可控方面的能力，中国特色因子。
F₆ = w₆₁×国产替代率 + w₆₂×供应链自主度 + w₆₃×关键技术掌握
w₆₁=0.5, w₆₂=0.3, w₆₃=0.2
门槛值：国产替代率≥30%
评分细则：替代率≥70%→10分 | 50%-70%→7分 | 30%-50%→4分
数据来源：行业研究报告 + 公司供应链披露
